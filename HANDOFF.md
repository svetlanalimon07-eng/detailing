## Как деплоить

**Пуш = деплой.** GitHub Pages автоматически публикует из корня ветки `main`.  
Обновление сайта занимает ~1 минуту после пуша.

**Хук настроен** — при любом изменении файла через Claude (Edit/Write)  
автоматически выполняется:
```bash
git add -A && git commit -m "auto: update site" && git push origin main
```
Вручную пушить не нужно — всё происходит само.

**Если хук не сработал** (например, сессия открыта не из папки проекта):
```bash
cd "/Users/a1/Сайты/siboil"
git add -A
git commit -m "описание изменений"
git push origin main
```

## Структура секций сайта (сверху вниз)

1. **Навигация** — фиксированная шапка, логотип `logo.jpg`, телефон, бургер-меню

   - `СТУДИЯ` — справа снизу, outline (slide up)
   - `ДЕТЕЙЛИНГ` — вертикально на границе фото, fade in
3. **О студии** — вводный текст, кнопка "Все услуги"
4. **Статистика** — 4 счётчика с анимацией: 7+ лет / 3500+ клиентов / 12 мастеров / 100% гарантия
5. **Услуги** — горизонтальный pin-scroll (GSAP), 4 слайда:
   - Полировка кузова (красный фон)
   - Керамическое покрытие (тёмный фон)
   - Бронирование PPF (тёмно-серый фон)
   - Химчистка салона (тёмно-красный фон)
6. **Наши работы** — карусель 6 карточек, touch/swipe
7. **Отзывы** — 6 карточек в стиле 2ГИС, рейтинг 4.9 / 87 отзывов
8. **CTA** — красный блок "Хотите узнать стоимость?", кнопка позвонить
9. **Контакты** — адрес, телефон, форма записи онлайн
10. **Footer** — логотип, ссылки, копирайт
