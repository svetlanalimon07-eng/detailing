# Siboil — Детейлинг студия Новосибирск

## Проект

Сайт детейлинг студии **Siboil Detail** (Новосибирск).  
Продакшн: **https://siboil-detailing.ru**

## Стек

- Чистый HTML/CSS/JS — один файл `index.html`, никакого сборщика
- GSAP 3.12.2 + ScrollTrigger (CDN) — горизонтальный скролл услуг, параллакс
- Google Fonts: Inter (основной шрифт)
- Локальный ассет: `logo.jpg` — логотип Siboil Detail (щит)

## Структура секций

1. **Hero** — две фото рядом, анимированные надписи SIBOIL / ДЕТЕЙЛИНГ / СТУДИЯ
2. **О студии** — вводный текст, кнопка
3. **Статистика** — 4 счётчика с анимацией
4. **Услуги** — горизонтальный pin-scroll (GSAP), 4 слайда
5. **Наши работы** — карусель 6 карточек с touch/swipe
6. **Отзывы** — 6 карточек в стиле 2ГИС, рейтинг 4.9
7. **CTA** — блок записи на диагностику
8. **Контакты** — адрес, телефон, форма
9. **Footer** — логотип, ссылки, копирайт

## Правила при редактировании

- Все стили — только inline в `<style>` внутри `index.html`
- Весь JS — только inline в `<script>` в конце `index.html`
- Цветовая гамма: **только чёрный, белый, красный** (`--red`, `--red-dk`)
- Никаких внешних зависимостей кроме GSAP CDN и Google Fonts
- После каждого изменения: `git add index.html && git commit -m "..." && git push origin main`
- Логотип `logo.jpg` не редактировать — это оригинальный файл бренда
